API Documentation

Welcome to the US Starbucks Locations API

  • When the User inputs a city and state this API will return addresses for all Starbucks stores located in that city. The returned object will be a list of JSON objects.
  • URL routes: /api/<city>/<state>
  • Method: Get
  • URL parameters: City and State. Both are required.
  • Error Resonses:
    • 404 Code: Page Not Found HTML opens.
    • 500 Code: 500 Error HTML opens.
  • The demo will return all Starbucks stores located in Miami, Florida
